Welcome to the Theater Ticket System.

The Theater Ticket System was the chosen semester project for the fall 2014 CS650 course @UAH. Since UAH likes multiple semester projects that perform requirements elicitation, design, develop, and test. I figured that Github would make a good centralized location to help maintain organization and change history for the future semesters.
